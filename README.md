### Prerequisites

- Python 3.7+
- pip (Python package manager)

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/ByteBanditKD/CryptoScriptHub.git
   ```
2. Navigate to the project directory:
   ```
   cd CryptoScriptHub
   ```
3. Install required packages:
   ```
   pip install -r requirements.txt
   ```
